# 零服务器成本的PWA视频站

## 关于前端

现在基本前端UI页也有了，页面基本参考抓取网站的html 手动保存版本在html/ 目录下，我们要做个支持者70k+视频的pwa网站，希望部署到cloudflare pages 并实现永久免费可运行。

## 关于数据

数据json基本在 \_by_id 和 \_by_tags，抓取的评论数据在 comments 目录下
用cloudflare的无服务器架构，m3u8保存到r2, 用户登录数据存客户端本地，评论数据做静态的，不允许用户评论先，用户数据完全不上传，比如喜欢，播放记
录等，默认直接注册成本分配一个用户名和UID，还有个默认avatar,
只有最后开通会员的时候，才收集他的email和开通会员时间，时长，付费金额...

## 关于部署

目前claude 已基本完善了架构，实现github actions 自动部署到 cloudflare
，本地可以测试前后端，接口测试支持TDD，推荐算法就用最简单的 最新是最新，推荐是热门，视频详情页的相关推荐就固定4个 分配机会需要尽量全覆盖到所有ID都有机会在别的视频详情页下出现。 尽量用静态 json存r2来解决减少服务器运算压力，首页的分类标签下的分
页加载也是固定的顺序，我们不考虑更新，也没有搜索功能，抖音刷的信息流也是按固定的分页加载，只要能支持用户一直加载更多页看下去即可。主要是想html css js json基本都命中 cloudflare的缓存，减少worker的计算用量，r2的成本控制在免费的10G以内，支持10K以上的用户日活还能保证 worker和pages的计算费用不超过每月5美金...
